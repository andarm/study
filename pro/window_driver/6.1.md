## 读写接口函数 

- 写数据
status = WdfRequestRetrieveInputBuffer(
                        Request,
                        sizeof(ULONG),
                        &inBuffer,
                        NULL
                        );

- 读数据 
status = WdfRequestRetrieveOutputBuffer(
        Request,
        sizeof(ULONG),
        &outBuffer,
        NULL
        );

## 读写返回值 

WdfRequestCompleteWithInformation(Request, status, sizeof(ULONG));
param1: 请求句柄
param2: 返回状态
prram3: 返回数据长度 
根据状态值，应用程序可判断读写是否成功。
