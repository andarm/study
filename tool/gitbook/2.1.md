# gitbook 发布到github 
 
## [目录](README.md) 

### 步骤 

- github 创建仓库 
- 然后对仓库，开启github pages 功能。

    开启可以是master方式： 这样子master存储的就是编译好发布内容。 
    开启gh-pages方式，将master分支作为markdown源代码保存，将gh-pages分支作为发布分支。 
    注意：分支必须是gh-pages的命名  

- gitbook中运行git serve 就会生成_book 文件夹，将文件夹内容推送到gh-pages分支即可 

- master分支，就是需要作为存储库
