# 添加C/C++相关的代码片段 
- **snippet 插件直接就有关于C/C++ 相关插件的使用，直接安装就可以使用** 
- 添加自定义的file 文件说明和函数说明 
```
"file decription": {
	"prefix": "file",
	"body": [
		"/********************************************************************",
		" * FileName    :                                                    *",
		" * Data        :                                                    *",
		" * Description :                                                    *", 
		" * Version     :                                                    *",
		" * Hardware    :                                                    *",
		" * Remarks     :                                                    *",
		" ********************************************************************/",
	],
	"description": "For file Description"
},
"Function decription": {
	"prefix": "file",
	"body": [
		"/********************************************************************/",
		"/*函数名称:                                                         */",
        "/*函数功能:                                                         */",
        "/*入口参数:                                                         */",
        "/*出口参数:                                                         */",
        "/*资源    :                                                         */",
        "/*子函数  :                                                         */",
        "/*更改日期:                                                         */",
        "/********************************************************************/",
	],
	"description": "For Function Description"
},

```
