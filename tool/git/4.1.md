## 4.1.git基本命令 

## [4.目录](./README.md)

#### 将远程库更新到本地库 

git fetch origin master


#### github 中branch 使用  

- 查看本地分支 
git branch 
- 查看远程分支 
git branch -r 

### 提交到远程库 

git push origin master 

#### 目录重命名 

git mv -f oldfolder newfolder
git add -u newfolder  

#### 目录删除 

git rm -r --cached .idea 

#### 本地回滚 

git reset --hard bb // 当前本地是aa版本，需要回滚bb版本
